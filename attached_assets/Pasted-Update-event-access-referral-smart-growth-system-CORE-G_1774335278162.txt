1. EVENT ELIGIBILITY LOGIC

Event Field:
- eligibleGender: "Male" / "Female" / "Both"

User Field:
- user.gender: "Male" / "Female"

Dashboard:
- Show ALL events to all users

Event Detail Logic:

If event.eligibleGender == "Both" OR event.eligibleGender == user.gender:
  → Show "Apply for Shift" button

Else:
  → Hide Apply button
  → Show "Refer & Earn" button

  - "Male Only"
  - "Female Only"
  - "Open for All"

  "This event is not available for your profile, but you can refer and earn."
